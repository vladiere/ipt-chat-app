## Description

Integrative Programming and Technologies Chat Application group project with socket.io, nodejs, mysql and quasar(vuejs framework).

### Installation

```bash
cd ipt-chat-app
npm install
quasar dev
```
## Backend dependencies installation and run 
```bash
cd ipt-backend
npm install
npm run dev
```
